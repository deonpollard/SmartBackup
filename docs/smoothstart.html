Smoothstart Guide

help you do you first backup in 3 easy steps

Step 1. get ready

You should be operational within 10-15 minutes!

  • Download and Run Installer - this will install SmartBackup
  • Run SmartBackup - a browser showing the operational console will appear



Install and Go

There are a number of ways to install SmartBackup. Choose the method that is best for your environment.

Starter Edition

This option uses an installer. It is is the fastest way to get SmartBackup up and running. If you're evaluating SmartBackup, use this option and try for free for as long as you like. Starter Edition does not have all the features, see Reference Guide. Do steps below using the Starter Edition to install..

Business Edition

This option uses an installer, and is the most straightforward way to get your production copy up and running. Do steps below after purchasing a business edition license.



Download the Installer

  1. Head to AcuWorkflow and download or buy a copy

Run the Installer

Run the installer. Follow the prompts to install SmartBackup. You'll be asked for the following info:

  1. Installation directory – this is where SmartBackup will be installed
  2. Full/Custom install – Always select full, unless you are upgrading/re-installing and want to preserve existing settings and the backup catalog
  3. SmartBackup will start up in your browser once installation is complete.

Setup SmartBackup

Once -off setup SmartBackup. Refer to the Setup Guide

  1. Vault - designate space within Smartsheet for backups
  2. Tenants – Provide some details for backup users
  3. Optionally setup schedules
